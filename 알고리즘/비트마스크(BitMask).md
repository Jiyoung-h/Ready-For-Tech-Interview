# 비트마스크(BitMask)

- 집합 요소들의 구성 여부를 표현할때 유용
- DP, 순열, 배열 활용만으로 해결할 수 없는 문제
- 작은 메모리와 빠른 수행시간으로 해결 가능 (원소의 수가 많지 않을때)
- 집합을 배열의 인덱스로 표현
```
[1,2,3,4,5] → 11111(31)
[2,3,4,5]   → 11110(30)
[1,2,5]     → 10011(19)
[2]         → 00010(2)
```

### 삽입
- OR 연산 : i번째 인덱스 삽입 (i번째 비트 값을 1로 변경)

  ```10101 || 1 << i```

### 삭제
- AND연산과 NOT 연산 

  ```11101 && ~1 << i```

### 조회
- AND연산 : 0이 아니면 1

  ```10101 && 1 << i```
